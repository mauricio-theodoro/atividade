# Projeto de Testes Unitários com Spring Boot

## Tecnologias
- Java 11
- Spring Boot
- JUnit 5
- Mockito
- MariaDB

## O que foi feito
- Implementação de CRUD para entidade `Usuario`
- Criação de testes unitários com `Mockito`
- Testes validados com `MockMvc`

## Como testar
- Rodar `./mvnw test`
- Testes cobrem criação e listagem de usuários
